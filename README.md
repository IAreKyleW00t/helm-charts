# ☸️ Helm Charts

My custom Helm Charts, hosted on GitHub Pages using the
[helm/chart-releaser-action](https://github.com/helm/chart-releaser-action)!

## Usage

```sh
helm repo add iarekylew00t https://iarekylew00t.github.io/helm-charts
helm repo update
```
